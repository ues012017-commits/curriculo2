⚠️ Não foi possível ler o arquivo

Ocorreu um erro ao extrair os textos do seu documento. Os motivos mais comuns são:

• O PDF está protegido por senha ou com restrições de cópia.
• O arquivo está corrompido ou em formato incompatível.
• O documento é uma imagem sem texto selecionável e nossa IA não conseguiu reconhecê-lo.
• O conteúdo é muito curto para ser um currículo completo.

💡 Dicas para resolver:

1. Se o PDF foi salvo como imagem (escaneado), tente abri-lo em outro editor e salvar como PDF normal com texto selecionável.

2. Salve seu currículo como .docx (Word) — nosso sistema lê Word com muita precisão.

3. Ou preencha os campos manualmente no nosso sistema — é rápido e fácil!

4. Se preferir, baixe nosso modelo em texto, preencha e tente importar novamente:
